# School Forum 接口对接文档

## 概述

本文档为 School Forum 社区论坛项目提供完整的前后端接口对接说明。前端基于 Vue 3 + Vite 技术栈构建，通过 RESTful API 与后端服务进行通信。

## 基础信息

| 项目 | 说明 |
|------|------|
| 服务基础URL | `http://localhost:8085` |
| API 前缀 | `/api` |
| 前端请求基础URL | `/api` (开发环境通过代理) |
| 认证方式 | Bearer Token (JWT) |
| 超时时间 | 10000ms |
| 数据格式 | JSON |

## 通用响应结构

### 成功响应
```json
{
  "code": 200,
  "message": "success",
  "data": { }
}
```

### 错误响应
```json
{
  "code": 500,
  "message": "错误信息",
  "data": null
}
```

## 认证说明

### 请求头
所有需要认证的接口需要在请求头中携带 Token：
```
Authorization: Bearer <token>
```

Token 存储在 `localStorage` 或 `sessionStorage` 中，前端会自动在请求拦截器中添加。

### 错误码处理
| HTTP 状态码 | 说明 | 处理方式 |
|-------------|------|----------|
| 401 | 未授权 | 清除 Token 并跳转到登录页 |
| 403 | 没有权限访问 | 提示用户权限不足 |
| 404 | 请求资源不存在 | 提示资源不存在 |
| 500 | 服务器错误 | 提示服务器错误 |
