# 论坛分类说明

## 导航菜单分类

| 分类 | Key | 说明 |
|------|-----|------|
| 学习 | study | 学术交流、课程讨论、学习资料分享 |
| 生活 | life | 校园生活、日常趣事、生活技巧 |
| 活动 | activity | 社团活动、校园赛事、聚会组织 |
| 求职 | career | 实习招聘、求职经验、职业规划 |
| 交易 | trade | 二手物品、书籍交易、闲置交换 |

## 分类详情

### 1. 学习 (study)
- 课程讨论与笔记分享
- 学术问题交流
- 学习资料共享
- 考试经验交流

### 2. 生活 (life)
- 校园日常分享
- 食堂美食推荐
- 宿舍生活
- 兴趣爱好交流

### 3. 活动 (activity)
- 社团活动发布
- 校园赛事信息
- 线下聚会组织
- 志愿活动招募

### 4. 求职 (career)
- 实习岗位推荐
- 招聘信息发布
- 面试经验分享
- 职业规划讨论

### 5. 交易 (trade)
- 二手书籍交易
- 闲置物品交换
- 数码产品转让
- 生活用品互助
